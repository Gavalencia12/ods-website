Accesibilidad y Personalización

Esta página te permite ajustar la visualización del sitio para mejorar tu experiencia según tus necesidades. A continuación, puedes activar o desactivar distintas opciones de accesibilidad:

Cuadro de prueba de accesibilidad

Este cuadro refleja los cambios en la visualización cuando activas las opciones de accesibilidad.

Por ejemplo, aquí puedes ver cómo cambia el tamaño del texto, los colores, el contraste, el cursor y otros elementos.

Este es un enlace de prueba para ver el efecto de "Quitar Links".

¿Para qué sirven estas opciones?

  • Modo Oscuro: Reduce la fatiga visual al cambiar a colores oscuros, ideal para ambientes con poca luz.
  • Alto Contraste: Mejora la visibilidad usando colores con alto contraste, facilitando la lectura a personas con visión limitada.
  • Aumentar / Disminuir fuente: Ajusta el tamaño del texto para facilitar la lectura según tus preferencias.
  • Cursor Grande: Aumenta el tamaño del cursor para facilitar su seguimiento visual.
  • Quitar Imágenes: Elimina imágenes para evitar distracciones o mejorar la velocidad de carga.
  • Quitar Links: Deshabilita los enlaces, útil para evitar clics accidentales y destacar mejor el contenido.

Filtros para daltonismo

Puedes elegir filtros de color para simular distintos tipos de daltonismo y adaptar los colores del sitio para una mejor percepción visual.
